# Automatický systém sběru dat pro dopravní průzkum

Systém, který sbírá, detektuje a klasifikuje dopravní značky s jejich GPS souřadnicemi pro webovou aplikaci, která zobrazuje statistiku výskytu, četnosti a druhu značky.
Aplikace je funkčně rozdělená na stranu serveru a zařízení, které je určeno pro sběr, v tomto případě se jedná o Raspberry Pi 4 B s GPS modulem a Arducam kamerou.

## Co dokumentuje?

Dokumentace bude určena pro oponenta, který bude přítomen při obhajobě bakalářské práce. Bude popisovat, jak dané zařízení vyzkoušet, aby došlo k úspěšné kontrole toho, že je systém funkční.
